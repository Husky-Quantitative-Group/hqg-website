# HQG Website

Official (in-prog) website for the Husky Quantitative Group - UConn's student-run quantitative fund.

## Quick Start

```bash
npm install
npm start
```

Opens at `http://localhost:3000`

## Deployment

```bash
npm run deploy
```

Deploys to GitHub Pages at `uconnquant.com`

## Commands

- `npm start` - Run development server
- `npm run build` - Build for production
- `npm run deploy` - Deploy to GitHub Pages
- `npm test` - Run tests
